# Tic-Tac-Toe

A simple and interactive Tic-Tac-Toe game built using HTML, CSS, and JavaScript. The game supports two players taking turns and automatically detects winners and draws.

## Features

* Two-player gameplay (X vs O)
* Winner detection
* Draw detection
* Turn indicator
* Restart Game functionality
* Responsive design for desktop and mobile devices
* Clean and modern user interface

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Play

1. Player **X** starts the game.
2. Players take turns selecting empty cells.
3. The first player to get three symbols in a row, column, or diagonal wins.
4. If all cells are filled and no player wins, the game ends in a draw.
5. Click **Restart Game** to start a new match.

## Game Rules

Winning combinations:

* Horizontal rows
* Vertical columns
* Diagonals
